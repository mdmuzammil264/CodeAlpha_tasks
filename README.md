# 🐍 Python Projects Collection

A collection of beginner-friendly Python projects built to practice core programming concepts such as loops, conditionals, dictionaries, file handling, object-oriented programming, and GUI development.

## 📌 Projects Included

1. 🎮 Hangman Game
2. 🤖 Rule-Based ChatBot
3. 📈 Stock Portfolio Tracker

---

# 🎮 1. Hangman Game

A feature-rich Python Hangman game with difficulty levels, a hint system, multiplayer mode, and a Tkinter GUI.

## Features

* Easy, Medium, and Hard difficulty levels
* Hint system (one hint per round)
* Multiplayer mode with scoring
* Console-based gameplay
* Tkinter GUI version
* ASCII art gallows

## Technologies Used

* Python
* Tkinter
* Random Module
* OOP Concepts

## Run

```bash
python hangman.py
```

## Concepts Learned

* Loops
* Conditional Statements
* Functions
* Classes & Objects
* Lists & Sets
* GUI Development with Tkinter

# 🤖 2. Basic Rule-Based ChatBot

A simple chatbot that responds to predefined user inputs using exact and partial keyword matching.

## Features

* Rule-based conversation system
* Exact keyword matching
* Partial sentence matching
* Interactive command-line interface
* Easy to customize

## Technologies Used

* Python

## Run

```bash
python chatbot.py
```

## Sample Conversation

```text
You: hello
Bot: Hi! How can I help you today?

You: bye
Bot: Goodbye! Have a great day!
```

## Concepts Learned

* Dictionaries
* Functions
* Loops
* String Manipulation
* User Input Handling
* Basic Chatbot Logic

# 📈 3. Stock Portfolio Tracker

A console-based stock portfolio tracker that calculates investment value using predefined stock prices.

## Features

* Hardcoded stock price dictionary
* Multiple stock entries
* Portfolio value calculation
* Portfolio summary table
* Optional CSV export

## Technologies Used

* Python
* CSV File Handling

## Run

```bash
python task2_stock_tracker.py
```

## Sample Output

```text
Stock      Qty      Price      Value
----------------------------------------
AAPL       10       $180       $1800
TSLA       5        $250       $1250
----------------------------------------
TOTAL                          $3050
```

## Concepts Learned

* Dictionaries
* Input/Output
* Arithmetic Operations
* Loops
* File Handling
